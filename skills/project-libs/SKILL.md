# Project Libraries Skills

| Skill | Description |
|-------|-------------|
| dspy-utils | Use when working with DSPy library, needing pre-configured language models, loading data into dspy.Example format, managing prompts in external files, or building DSPy modules with the prompt-as-files pattern |
| offchain-lib | Use when working with the offchain-lib, a library for working with off-chain data, such as APIs, databases, and files. Includes batching logic for rpc calls. |
| sequence-classification-llm | Use when fine-tuning or training text classification/regression models with HuggingFace transformers. Triggers on tasks involving sentiment analysis, text categorization, scoring predictions, or any sequence-to-label modeling. |
