# DevOps Skills

A collection of skills for DevOps practices, deployment automation, and infrastructure management. Each skill focuses on a specific aspect of building and deploying production systems.

## Activation Triggers

- Writing shell scripts for automation or CI/CD
- Deploying applications to cloud platforms
- Managing secrets and credentials securely
- Setting up production deployment pipelines
- Configuring Railway services or databases
- Implementing fault-tolerant automation
- Handling sensitive environment variables
